## How to Run Locally

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the Flask app: `python app.py`
4. Open your browser to `http://127.0.0.1:5000`

## How to Use

1. Click on any planet to select it
2. Enter mass in kilograms
3. Click "Calculate Force" to see the gravitational force in Newtons
4. The result shows F = ma calculation with the planet's gravity
